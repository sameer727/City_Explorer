## ⚡ Quick Commands

### Setup
```bash
# Backend
cd backend && npm install && npm start

# Frontend (new terminal)
cd frontend && npm install && npm run dev
```

### Development
```bash
# Start
npm start              # backend
npm run dev           # frontend

# Build
npm run build         # frontend only

# Clean
npm audit            # check security
npm update           # update packages
```
