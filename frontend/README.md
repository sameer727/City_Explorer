# CityExplorer Frontend

A premium, modern React frontend for CityExplorer.

## Setup

1. Install dependencies:
   npm install
2. Copy environment file:
   cp .env.example .env
3. Start dev server:
   npm run dev

Backend should run at `http://localhost:5000`.

## Implemented Pages

- Home
- Explore Cities
- Attractions
- Place Details
- Favorites
- Login

## Tech

- React + Vite
- TailwindCSS
- Framer Motion
- React Router
- Axios
